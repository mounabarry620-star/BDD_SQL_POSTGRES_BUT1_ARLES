Exercice 4

Considérez la structure suivante :

struct Fiche { string nom,prenom; double taille; bool fille; string commentaire; string image;};

Elle représente une fiche personnelle simplifiée permettant de constituer un annuaire. Les champs permettent de mémoriser le nom, le prénom, la taille en mètre, un booléen qui indique si la personne est une fille, un commentaire textuel libre et un nom de fichier contenant une photo de la personne.

Téléchargez le fichier [exo4.zip] et décompressez-le dans votre répertoire de travail. Il contient, dans un répertoire « exo4 », un exemple initial de fichier texte d'annuaire ainsi que quelques photos. Ouvrez le fichier « MonAnnuaire.txt » pour étudier sa structure. Pour chaque fiche, vous trouverez :

une ligne entière comportant le nom de la personne ;

une ligne entière comportant le prénom de la personne ;

sur une même ligne, la taille en mètre (un réel) puis un caractère H ou F suivant que la personne est un homme ou une femme ;

une ligne formant la totalité du commentaire ;

une ligne comportant le nom du fichier d'image servant de photo d'identité.

Ouvrez le squelette de programme « exo4.cpp » fourni. Vous devez remplir les fonctions contenant des points de suspension afin qu'il permette de charger un annuaire en mémoire à partir de son fichier. Exportation au format CSV

Le format CSV pour Comma-separated Values (cf. Wikipédia) (valeurs séparées par des virgules) est un format très simple pour exporter des données sous forme de tableau. Un fichier respectant cette structure peut facilement être importé ou exporté dans Excel ou LibreOffice. Vous devez savoir écrire (et si possible relire) par un programme un tel format pour le jour où vous en aurez besoin !

Un fichier CSV, comme le fichier « exemple1.csv » qui se trouve dans exo4.zip que vous avez téléchargé, contient un ensemble de lignes de texte. Chaque ligne du fichier correspond à une ligne dans votre tableur. Les valeurs qui vont dans chaque colonne du tableur sont séparées dans le fichier par un caractère virgule (','). Les valeurs numériques sont écrites au format anglo-saxon (avec un point comme séparateur décimal et non une virgule). Les chaînes de caractères doivent être écrites entre guillemets anglaises : « " ». Pour obtenir ce caractère dans une chaîne C++, vous devez le protéger avec un backslash : « "Voici un caractère guillemet \" dans une chaine C++ !" ».

Pour donner les titres des colonnes, il n'y a pas de subtilité : il suffit d'écrire en premier, dans le fichier, une première ligne avec les titres textuels (donc chacun entre guillemets anglaises, séparées par des virgules). Merci d'utiliser (exactement) les titres suivants : « Nom », « Prénom », « Taille », « Genre », « Commentaire » et « Image ». Pour indiquer le genre de la personne, vous écrirez comme valeur soit la lettre « H », soit la lettre « F ».
